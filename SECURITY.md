# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in WritePay, please report it responsibly.

**Email**: security@writepay.xyz

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and provide a fix timeline.

## Scope

- Smart contract logic (`contracts/sources/writepay.move`)
- Encryption key management (`app/api/encryption/`)
- Authentication and access control

## Out of Scope

- Issues in third-party dependencies (report upstream)
- Social engineering attacks
- Denial of service attacks
